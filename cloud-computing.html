What is Cloud Computing? Beginner Guide

Published by StudentTechLK • Beginner Friendly

Cloud Computing means using internet-based services like servers, storage, databases, software and networking instead of using only your own computer.

Simple example ekak balamu. Oya Google Drive use karanawanam, oyage files internet eke save wenawa. Oya phone eka change kalath, Gmail account eken login wela files tika aye ganna puluwan. Eka cloud service ekak.

Why is Cloud Computing Important?

Companies cloud computing use karanne loku computer systems buy karanna one nathi nisa. Cloud platforms use karala companies walata websites host karanna, apps run karanna, data save karanna and security manage karanna puluwan.

Popular Cloud Platforms

  • Amazon Web Services - AWS
  • Microsoft Azure
  • Google Cloud Platform - GCP
  • Oracle Cloud

Cloud Computing Examples

Api daily life eke cloud services godak use karanawa. Google Drive, Gmail, Netflix, YouTube, WhatsApp backup, online games, and websites godak cloud servers use karanawa.

Cloud computing learn karana students lata future eke good job opportunities thiyenawa.

  • Cloud Engineer

  • Cloud Support Engineer
  • System Administrator
  • Solutions Architect

Beginner Learning Path

Cloud computing igena ganna beginner kenek me basics tika first learn karanna one.

  1. Computer networking basics
  2. Linux basics
  3. Server basics
  4. Database basics
  5. Security basics
  6. AWS or Azure beginner services

StudentTechLK Tip

Beginner kenek nam first AWS Free Tier or Microsoft Azure free account ekak use karala small projects hadanna. Theory witharak madi. Practical projects karanna.

Final Words

Cloud Computing kiyane future IT field ekak. Oya student kenek nam, danma basics tika igena ganna patan ganna. Small projects hadala GitHub portfolio ekata add karanna.
